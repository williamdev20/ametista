# Funções

## Função sem retorno
```
fun void welcome() do
    out "Hello, World!"
end
```

## Função com retorno
```
fun str sayHello() do
    return "Hello!"
end
```

## Função sem parâmetro
```
bool isNumber() do
    return false
end
```

## Função com parâmetro
```
int somar(int x, int y) do
    return x + y
end
```
