# Variáveis

Ametista possui os tipos **int, float, str, bool, list, map**
### Números

#### Inteiro

```
int numero = 45
```

#### Ponto flutuante

```
float valor = 0.0
```

### Textos

```
str linguagem = "Ametista"
```

### Booleanos

```
bool status = true
```

### Arrays

```
list<str> frutas = ["maçã", "banana", "goiaba"]
```

### Hashmaps

```
map<str, int> numerosExtenso = {"zero": 0, "um": 1, "dois": 2}
```


## Tipagem inferida

```
var text = ":D"
```


## Constantes
```
const str linguagem = "Ametista"
```
